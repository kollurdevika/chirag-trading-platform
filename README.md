# Chirag Trading Platform

A full-stack trading platform built with:

- Node.js
- Express.js
- MongoDB
- Next.js

## Features

- User Authentication
- Trading Dashboard
- Order Management
- Portfolio Tracking

## Installation

### Backend

```bash
cd backend
npm install
npm start

cd frontend
npm install
npm run dev
